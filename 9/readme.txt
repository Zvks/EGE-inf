Если задача выяснить, какое количество троек чисел может являться сторонами треугольника, 
то есть удовлетворять неравенству треугольника.
То сначала надо провверить каждую из сторон на соответствие неравенству треугольника,
первый столбец сумма первого и второго чисел тройки
второй столбец если сумма первого и второго чисел тройки > третьего числа то 1, иначе 0
третий столбец сумма первого и третьего чисел тройки
четвертый столбец сумма первого и третьего чисел тройки > второго числа то 1, иначе 0
пятый столбец сумма второго и третьего чисел тройки
шестой столбец сумма второго и третьего чисел тройки > первого числа то 1, иначе 0
седьмой столбец если второй, четвертый и шестой столбцы 1 то 1, иначе 0
сумма седьмого столбца

Автоматическая фотокамера производит растровые изображения размером 768 на 600 пикселей. 
При этом объём файла с изображением не может превышать 450 Кбайт, упаковка данных не производится. 
Какое максимальное количество цветов можно использовать в палитре?

Решение.
Объём растрового изображения находится как произведение количества пикселов в изображении на объём памяти x, 
необходимый для хранения цвета одного пиксела: 768 * 600 * x ≤ 450 * 8 * 1024 бит, откуда x = 8 бит. 
Значит, в изображении можно использовать не более 2**8 = 256 цветов.
